# Technologies
- React TypeScript
- GSAP Animation
- Node.js
- Jest
- React Testing Library

# Installations
- Fork or clone this repository.
- `cd` into `frontend` directory.
- Run `npm i` to install necessary dependencies.
- Run `npm run dev` to start the app!
